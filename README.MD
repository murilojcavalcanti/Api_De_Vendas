## Description [.NET] https://learn.microsoft.com/pt-br/dotnet/framework/

Esse projeto foi criado com o objetivo de controlar as vendas de vendedores onde, voce adiciona os vendedores e produtos ao banco de dados,
apos isso voce ira adicionar as vendas que terao o status padrao de aguardando pagamento e a data da venda, e você deverá adicionar o id do 
vendedor e dos produtos daquela venda.
## Rodar APP no VSCode
Antes de rodar e preciso configurar os seus secrets com as informacoes de banco de dados.

Para rodar o projeto use o terminal e rode o seguinte comando:

``` bash

#Rodar Projeto
